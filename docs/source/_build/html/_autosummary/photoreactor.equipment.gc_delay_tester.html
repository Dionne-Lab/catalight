photoreactor.equipment.gc_delay_tester

Module is used to test the response time of the gas chromatograph.

The GC and gas system are used to check the response time of the system. The user must make edits to this module in the current iteration. Update the delay times, flow rates, and gas details to test for. The system will flow gas, then wait the input delay time until starting the GC. This allows the user to check how long the it takes for the gas supply to reach steady state to the GC. For data analysis, see the “run_plot_chromatograms_stacked.py” module.
