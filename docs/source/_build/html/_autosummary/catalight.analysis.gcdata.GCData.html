Find the area under the peak using a trapezoidal method.

Calculate for each peak identified using bounds from integration_inds. Integrates peak trapezoidally in units of seconds*signalintensity.

Returns:

counts – counts rounded using np.around()

Find bounds of integration for apexes.

Edge values determined by self._half_index_search. Output in the same (time) order as the apex_ind array with the location (index) of the left and right points for integration

Returns:

  • left index (numpy.ndarray of int) – Left bounds of integration.

  • right index (numpy.ndrray of int) – Right bounds of integration.
