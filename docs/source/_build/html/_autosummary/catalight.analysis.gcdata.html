catalight.analysis.gcdata

Gas chromatograph data and associated processing methods.

GCData is meant to represent a gas chormatograph data file and associated data processing techniques. Individual data files are operated on by this class, then processed data for an experiment is compiled, analyzed, and plotted elsewhere.

Classes

A class for processing/analyzing gas chromatograph data from an ASCII file.
