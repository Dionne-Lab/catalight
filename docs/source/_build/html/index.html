Welcome to photoreactor’s documentation!

The purpose of this project is to share tools developed in the Dionne Lab at Stanford University for automating photocatalysis experiments and data analysis.

If your experimental setup looks anything like this:

You might be able to take advantage of our code base to accelerate your experiments!!

Don’t have this exact configuration? We’ve designed projectname with modularity in mind. We want to enable interested labs to develop their own equipment drivers to and reuse as much of our code as possible. Please visit the Development Guide for more information about customizing projectname for your applications!
